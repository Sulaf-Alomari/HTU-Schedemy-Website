# **App Name**: Sci-Ed Schedule

## Core Features:

- Dashboard Summary Cards: Displays summary cards for total courses, instructors, rooms, and scheduled classes on the dashboard page.
- Weekly Timetable View: Displays a weekly calendar/timetable view in a grid format (days Ã— time slots) on the dashboard page.
- Course Management Table: Manages courses with a table listing Course Code, Name, Credit Hours, and Group, including Add/Edit/Delete functionalities.
- Conflict Resolution Engine: Automatically resolve scheduling conflicts between courses, instructors, TAs, rooms and times.
- Add Timetable Entry: Allows users to assign a course with the instructor, TA, room, and time slot; validates conflicts before adding to the timetable.

## Style Guidelines:

- A calm blue (#3498db) for the header and sidebar to establish a professional tone.
- Light gray (#f2f2f2) for backgrounds and card containers to ensure content stands out.
- A vibrant green (#2ecc71) to highlight active states and interactive elements, providing visual cues to users.
- Responsive grid layout to adapt seamlessly to different screen sizes.
- Consistent use of Font Awesome or Material Design icons.
- Subtle transitions and animations for interactive elements (e.g., hover effects on buttons, smooth modal openings).
